Install softwares

Wagman firmware installation

In order to manage nodecontroller and extension_node, wagman has to run its management program. Download the latest firmware here to your computer.

Unzip the firmware:

unzip <file-name>
# if you do not have unzip, install it using sudo apt-get install unzip

Install avrdude to upload the firmware:

sudo apt-get install avrdude

Connect microUSB-USB cable to wagman with the computer.


Run install.sh:

cd wagman_init
./install.sh

Nodecontroller and extension_node firmware installation

**WARNING: before procceding the following steps, pull out wagman's power adapter plug.

Follow the instruction here to upload the latest version of both nodecontroller and extension_node firmware. As a result, you will have two micro SD cards for both devices, respectively.

Insert each micro SD card on the designated device (nodecontroller firmware for nodecontroller and extension_node firmware for extension_node). eMMC card should also be inserted on both devices.
