class external_communicator.pika_push(group=None, target=None, name=None, args=(), kwargs={})¶

A pika client for rabbitMQ to push messages to the cloud. When the cloud is connected, the pull client sends pull requests and puts messages into the outgoing queue. When the outgoing queue is not empty, this pika client will connect and push those messages to the cloud.

class external_communicator.pika_pull(group=None, target=None, name=None, args=(), kwargs={})¶

A pika client for pulling messages from the cloud. If messages are sent from the cloud, this client will put them into the incoming queue.

class external_communicator.external_client_pull(group=None, target=None, name=None, args=(), kwargs={})¶

A client process that connects to the data cache if the cloud is currently connected. Once a message is recieved, it is put into the outgoing queue.

class external_communicator.external_client_push(group=None, target=None, name=None, args=(), kwargs={})¶

A client process that connects to the data cache and pushes incoming messages.
