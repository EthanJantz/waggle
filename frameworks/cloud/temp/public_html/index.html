To use WAggLE V0.1 either download and run the following script, or follow the directory further below.

GetWAggLE.sh
Don't forget to fix your PYTHONPATH after runnnig the script!


Dependancies:
Pika: https://pypi.python.org/pypi/pika
PySensors: https://pypi.python.org/pypi/PySensors/

Setup:
  1. Grab the common tarball located here.
  2. Extract the tarball (I will use "tar -xvf waggle_commons-1.tar.gz")
  3. Enter the directory created from the tarball (waggle_commons-1/)
  4. Run "sudo make install"
    • This currently requires sudo - it will put an egg in your python directory
You should now have a working copy of the waggle_commons library.


Run:
  1. Grab the client tarball located here.
  2. Extract the tarball (I will use "tar -xvf waggle_node_controller-1.tar.gz")
  3. Enter the directory created from the tarball
  4. Simply run sensor_data.py and a data point will be sent to the cloud
    • Upon execution a link will be presented, you can find your data there
