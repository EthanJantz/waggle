Release Notes

  • Python Modules
    • Any modules installed can be accessed under the name waggle_commons

  • Data Collection
    • Any data collected is available to anyone with the address that is presented during installation
    • The data that is collected is from internal sensors on your machine, these sensors typically measure temperature and fan speeds, as well as other operation related data
    • The data is sent to the destination in a standard TCP/IP packet

  • Data Privacy
    • There should currently be no expected privacy of any data collected
